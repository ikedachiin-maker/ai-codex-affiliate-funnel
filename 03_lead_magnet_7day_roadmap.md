# 無料特典: CodexでAI副業ファネルを作る7日間ロードマップ

## 表紙コピー

CodexでAI副業ファネルを作る7日間ロードマップ

作業を増やすAI活用から、導線を作るAI活用へ。

## はじめに

AI副業でつまずく人の多くは、ツール選びで止まっています。

ChatGPT、Claude、Codex、画像生成、動画生成。どれも便利です。けれど、便利なツールを触れるだけでは売上にはなりません。

必要なのは、次の流れです。

```text
誰に届けるか
何を渡すか
どこで信頼を作るか
何を紹介するか
どう改善するか
```

このロードマップでは、Codexを「作業を代わりにやってくれる相棒」として使いながら、X、LINE、記事、Brainアフィリエイトまでの小さな導線を作ります。

## Day 1: Codexで作るものを決める

最初に決めるのは、記事タイトルではありません。作るべきはファネルです。

今回の基本形:

```text
X投稿
↓
LINE登録
↓
無料特典
↓
Codex教材
↓
マーケティング教材
```

今日やること:

- 発信テーマを1つ決める
- 誰に届けるかを決める
- 紹介する教材を2つ決める

おすすめテーマ:

```text
非エンジニア向け Codex/AI自動化で副業導線を作る方法
```

## Day 2: ペルソナを決める

ペルソナは、ふんわりした属性ではなく「今どこで止まっている人か」で決めます。

今回の読者:

- ChatGPTは触った
- Codexは気になる
- AI副業にも興味がある
- でも何を作ればいいか分からない
- 高額スクールは怖い
- まず低単価教材で学びたい

今日やること:

```text
この人が検索しそうな言葉を10個書く
この人がXで反応しそうな悩みを10個書く
この人が買う前に不安になることを10個書く
```

## Day 3: 無料特典を作る

無料特典は、豪華さより「次の行動につながること」が大事です。

おすすめ特典:

```text
CodexでAI副業ファネルを作る7日間ロードマップ
```

特典の役割:

- LINE登録の理由を作る
- Codex教材への自然な入口を作る
- ツールだけでなく導線設計が必要だと伝える

## Day 4: Codex教材への導線を作る

最初に紹介するのは、マーケティング教材ではなくCodex教材です。

理由:

- 読者の顕在欲求は「Codexを使えるようになりたい」
- いきなりマーケティング教材を出すと飛躍がある
- まず作業のハードルを下げると、その後の教育が入りやすい

紹介文の型:

```text
Codexを触りたいけど、何から始めるか分からない人は、まず全体像を掴むのが早いです。
僕も最初にここを飛ばすと、何を自動化すべきかがぼやけると思っています。
```

CTA:

```text
Codexアプリの教科書を見る:
[CODEX_BRAIN_AFFILIATE_URL]
```

## Day 5: Codexから明鏡へ橋を架ける

ここが一番重要です。

Codexを使えるようになると、記事、投稿、LP、LINE文面は作れます。でも「何を誰にどう届けるか」が決まっていなければ、作業が増えるだけです。

橋渡しメッセージ:

```text
Codexは作業を自動化する道具。
明鏡は、何を誰にどう届けるかを決める設計図。
```

## Day 6: X投稿を作る

X投稿は、教材を売る場所ではなく、悩みを言語化してLINEに送る場所です。

投稿テーマ:

- Codexでできること
- AI副業で失敗する理由
- ツールより導線が大事
- 作業自動化とマーケティング設計の違い
- Brain教材の選び方

CTA:

```text
この流れを7日で作るロードマップを無料で配っています。
[LINE_OPTIN_URL]
```

## Day 7: 改善サイクルを作る

見ないといけない数字は、フォロワー数だけではありません。

毎週見る数字:

- X投稿数
- インプレッション
- プロフィールクリック
- LINE登録
- Codex教材クリック
- Codex教材成約
- 明鏡クリック
- 明鏡成約

改善の順番:

```text
LINE登録が少ない → 無料特典と固定ポストを改善
Codex教材クリックが少ない → LINE 3-4通目を改善
明鏡クリックが少ない → ブリッジ教育を改善
成約が少ない → レビュー記事とFAQを改善
```

## 最後に

AI副業で大事なのは、AIに全部任せることではありません。

AIで作業を減らし、人間は「誰に、何を、どんな順番で届けるか」を考える。

この分担ができると、Codexは単なる便利ツールではなく、収益導線を作る作業台になります。
