# リンク差し替えシート

## 必須

| Placeholder | 入れるもの |
|---|---|
| `[LINE_OPTIN_URL]` | LINE登録URL |
| `[LEAD_MAGNET_URL]` | 無料特典の閲覧URL |
| `[CODEX_BRAIN_AFFILIATE_URL]` | Codexアプリの教科書の紹介リンク |
| `[MEIKYO_BRAIN_AFFILIATE_URL]` | 明鏡の紹介リンク |
| `[DISCLOSURE_TEXT]` | 本記事には広告・紹介リンクを含みます。 |
| `[YOUR_X_HANDLE]` | XアカウントID |
| `[YOUR_NAME]` | 発信者名 |

## UTM例

### Codex教材

```text
[CODEX_BRAIN_AFFILIATE_URL]?utm_source=x&utm_medium=social&utm_campaign=codex_front
```

### 明鏡

```text
[MEIKYO_BRAIN_AFFILIATE_URL]?utm_source=line&utm_medium=step&utm_campaign=meikyo_backend
```

## 公開前に確認

- Brainの紹介リンクが正しく発行されている
- 紹介率が最新である
- 商品価格が最新である
- PR表記が入っている
- LINE登録リンクが機能する
- 無料特典URLが非公開になっていない
