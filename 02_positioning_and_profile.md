# Xポジションとプロフィール

## 発信ポジション

AIツール紹介者ではなく、Codexを使って「副業ファネルの作業工程」を実際に組み立てる人。

## 肩書き候補

1. CodexでAI副業ファネルを作る人
2. 非エンジニア向けAI自動化メモ
3. Codex x 発信導線の実験室
4. AIエージェントで副業導線を作る人

おすすめ:

```text
CodexでAI副業ファネルを作る人
```

## プロフィール案

```text
CodexでAI副業ファネルを作る人。非エンジニア向けに、X投稿、note、LINE、Brainアフィリ導線を実装しながら発信。AIで作業を減らし、マーケティングで売れる流れを作る。無料ロードマップは固定ポストから。
```

## 固定ポスト

```text
AI副業で最初に作るべきものは、記事でも教材でもありません。

先に作るべきは「流れ」です。

Xで集める
LINEに登録してもらう
無料特典で信頼を作る
Codexで作業を自動化する
最後に、売れる導線へつなげる

この流れを7日で作るロードマップをまとめました。

無料で受け取る:
[LINE_OPTIN_URL]

[DISCLOSURE_TEXT]
```

## プロフィール下の導線

固定ポストとプロフィールリンクは必ずLINE登録へ統一します。Brainリンクを直接プロフィールに置くと、教育前の読者を取りこぼしやすいです。
