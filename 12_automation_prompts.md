# Codex Automations用プロンプト

## Daily X Drafts

スケジュール: 毎日 08:00

目的: X投稿案を毎日3本作る

プロンプト:

```text
ai-codex-affiliate-funnel のファネル設計を前提に、今日投稿するX投稿案を3本作ってください。

条件:
- 1本目: Codex実践ノウハウ
- 2本目: AI副業の勘違い
- 3本目: LINE登録への自然な誘導
- 誇張表現を避ける
- 商品リンクは直接貼らない
- 必要なら [LINE_OPTIN_URL] をCTAに使う

出力:
- 投稿本文
- 狙い
- 想定読者
- 注意すべき表現
```

## Weekly Funnel Review

スケジュール: 毎週月曜 09:00

目的: 先週の数字から改善案を出す

プロンプト:

```text
ai-codex-affiliate-funnel/13_kpi_tracking.csv を前提に、先週のファネル数字をレビューしてください。

見る数字:
- X投稿数
- インプレッション
- プロフィールクリック
- LINE登録
- Codex教材クリック
- Codex教材成約
- 明鏡クリック
- 明鏡成約

出力:
- 良かった点
- ボトルネック
- 今週改善する投稿テーマ
- LINE文面の改善案
- レビュー記事の改善案
```

## Monthly Report

スケジュール: 毎月1日 08:00

目的: 月次収益レポートを作る

プロンプト:

```text
ai-codex-affiliate-funnel/13_kpi_tracking.csv を前提に、前月の月次レポートを作ってください。

出力:
- 総インプレッション
- LINE登録数
- Codex教材成約数
- 明鏡成約数
- 推定報酬
- 改善すべき3点
- 来月の重点テーマ
- 次に作るべき記事/投稿/LINE案
```

## Compliance Guard

スケジュール: 公開前に手動実行

目的: 投稿・記事・LINE文面の公開前チェック

プロンプト:

```text
以下の文章を、Brainアフィリエイト公開前の観点でチェックしてください。

観点:
- 広告/紹介リンク表記が必要な箇所
- 稼げる保証に見える表現
- 景表法上の誇張表現
- ステマに見える表現
- Brain有料本文の転載に近い表現
- 読者に誤解を与える表現

出力:
- 修正必須
- 修正推奨
- そのままでよい
- 修正版

対象文章:
[ここに貼る]
```
